Large Language Models: A Guide for Humanities Scholars

A practical guide for humanities scholars

Last updated: February 2026

This guide aims to help you make informed decisions about whether and how to engage with Large Language Models. It is not advocacy for adoption. Some colleagues will find these tools useful for specific tasks; others will reasonably conclude they are not worth the trade-offs. Both positions are defensible.

The guide is designed for self-paced exploration. Start wherever suits your situation.

Choose a path

Each path takes you through the pages most relevant to your situation, in a suggested order. Time estimates assume focused reading.

I teach next term  (30–45 min)

Briefing → Getting Started (try the exercise) → Use Cases (teaching tasks) → Mindset → Rules (disclosure template for students)

I’m writing or researching  (45–60 min)

Briefing → Getting Started → Ancient Languages or Use Cases (research tasks) → Mindset → Rules (verification ladder, citation checklist) → Agentic AI (optional)

I need to use institutional tools  (20–30 min)

Copilot → Rules (data boundaries) → Mindset → FAQ

Or browse the full guide below — each page is designed to stand on its own.

The guide

Briefing

What LLMs are, what they aren't, ethical dimensions, and why informed awareness matters regardless of whether you use these tools. The foundation for everything else in this guide.

Getting Started

A 15-minute hands-on exercise, practical setup guidance (platforms, privacy settings), and core prompting principles. The best starting point if you want to experiment.

Mindset

AI requires a different approach from other technology. Foundational shifts in how to think about these tools, and practical techniques for getting better results — most of which draw on skills you already have.

Rules, Risks & Verification

The essential reference: data boundaries, a three-tier verification ladder, provenance and disclosure guidance, and copy-paste templates for system prompts, citation checks, and audit records.

AI Decision Sheet

A one-page printable reference for deciding whether and how to use AI for a specific task. Covers data boundaries, verification tiers, and disclosure — condensed from the Rules page.

Use Cases by Time Investment

Practical applications organised by how much time they require — from 15-minute tasks to deeper explorations. Each includes risks to watch for.

Ancient Languages and Texts

How LLMs handle translation, textual criticism, fragmentary evidence, and sacred texts. Worked examples and methodology for any language you work with.

Agentic AI

How AI is moving from chat to autonomous tools that plan, code, and execute multi-step tasks. What agentic tools are, how to use them, and what has been built with them.

Microsoft Copilot

Best practice for your institution’s default AI tool. What Copilot does well, where it falls short, and how to get the most out of it — including when to use other tools instead.

AI Bootcamp

Ten structured experiments that build advanced AI fluency for research, teaching, and public scholarship. Each takes 2–4 hours and produces a reusable tool or workflow. A practical companion to the rest of this guide.

Frequently Asked Questions

Direct answers to common concerns: replacement anxieties, student cheating, ethical questions, data protection, and more.

Glossary

AI terminology explained without unnecessary technicality. A reference to consult when you encounter unfamiliar jargon.

Resources

Curated reading list for humanities academics: sector guidance, research integrity, source-facing AI tools, pedagogy, and critical literacy.

About this guide
