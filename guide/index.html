Large Language Models: A Guide for Humanities Scholars

A practical guide for humanities scholars

Last updated: February 2026

This guide aims to help you make informed decisions about whether and how to engage with Large Language Models. It is not advocacy for adoption. Some colleagues will find these tools useful for specific tasks; others will reasonably conclude they are not worth the trade-offs. Both positions are defensible.

If you feel apprehensive about AI, you are in good company. Many colleagues share that feeling. This guide assumes no prior experience and no particular enthusiasm. The first steps are genuinely easy — a 15-minute exercise you can do right now — and the AI itself will help you learn how to use it. You do not need to become technical. You just need to decide, on your own terms, whether any of this is useful in your work.

Choose a path

Pick the route that fits your situation. Time estimates assume focused reading.

I teach next term  (30–45 min)

Briefing → Getting Started (try the exercise) → Use Cases (teaching tasks) → Mindset → Rules (disclosure template for students)

I’m writing or researching  (45–60 min)

Briefing → Getting Started → Ancient Languages or Use Cases (research tasks) → Mindset → Rules (verification ladder, citation checklist) → Agentic AI (optional)

I need to use institutional tools  (20–30 min)

Copilot → Rules (data boundaries) → Mindset → FAQ
